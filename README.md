# Airline-reservation-and-dynamic-seat-allocation-
This project demonstrates core Object-Oriented Programming concepts in C++ including function overloading and operator overloading. The library manages flight information, seat booking, cancellation, and automatic seat allocation across Economy, Business, and First class categories.
Language and Tools
Built with C++ using Object Oriented Programming concepts and 
developed in Visual Studio using C++17 standard.

Function Overloading
Implements 3 versions of initialize() for default setup, basic 
flight information, and full flight setup with seat layout.

Operator Overloading
Supports merging flights with +, removing seat class with -, 
scaling fares with *, comparing flights with == and <, 
and formatted output with <<.

Seat Management
Dynamic seat allocation across Economy, Business and First class 
with real-time availability tracking and automatic seat assignment.

Fare System
Fare calculation with different multipliers per seat class.
Economy is base fare, Business is 2.5x and First class is 4x.

Academic Information
Submitted as an Object Oriented Programming course assignment
using C++ class design, header files and separate main file.
